ajax-upload
===========

Una herramienta para subir archivos con ajax y manejar la propiedad progress

Soporte
-------
Navegadores que soportan esta herramienta:

 - IE	                   9
 - Firefox	             4
 - Chrome	               7
 - Safari	               5
 - Opera	               12
 - iOS Safari	           5
 - Android Browser	     5
 - Blackberry Browser	   7
 - Opera Mobile          12	
 - Chrome for Android	   31
 - Firefox for Android   25	
 - IE Mobile             10
